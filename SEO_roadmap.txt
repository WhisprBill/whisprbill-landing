## **PHASE 1: Technical Foundation (Week 1-2)**

### 1. **Google Search Console Setup** [searchengineland](https://searchengineland.com/guide/google-search-console-guide)

**Step-by-step:**

1. **Create Property**
   - Go to https://search.google.com/search-console
   - Click "Add Property"
   - Choose "Domain" property (covers all subdomains/protocols)
   - Enter: `whisprbill.com`

2. **Verification Methods** (choose one):
   - **DNS Method (RECOMMENDED)**: Add TXT record to GoDaddy DNS
     - Most comprehensive
     - Doesn't require touching site code
     - Verifies all protocol variations
   - Alternative: HTML file upload to `/public/`
   - Alternative: Meta tag in `<head>`

3. **Submit Sitemap**
   - Navigate to "Sitemaps" in left menu
   - Add: `https://whisprbill.com/sitemap.xml`
   - Submit and verify "Success" status

4. **Link Google Analytics**
   - In Analytics, go to Admin → Property Settings
   - Select "Search Console" → "Link"
   - Choose matching property
   - Check monthly that connection remains active
